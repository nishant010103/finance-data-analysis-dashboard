# 📊 Finance Data Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing financial investment data to extract meaningful insights using **Python, Pandas, NumPy, and Plotly**.

The goal is to help businesses understand:

* Sector-wise investments
* Risk vs Return patterns
* ROI (Return on Investment)
* Monthly trends
* Profitability insights

---

## 🚀 Tech Stack

* Python 🐍
* Pandas
* NumPy
* Plotly (Interactive Dashboard)

---

## 📊 Key Analysis Performed

### 1. Sector-wise Investment Analysis

* Identifies which sector has the highest investment

### 2. Investment Type Performance

* Compares returns from Stocks, Mutual Funds, Bonds, ETFs

### 3. Risk vs Return Analysis

* Understand relationship between risk level and returns

### 4. ROI Analysis

* ROI = Return / Investment
* Helps evaluate efficiency of investments

### 5. Monthly Trend Analysis

* Tracks return performance over months

### 6. Profit vs Loss Analysis

* Identifies top-performing investments

---

## 📈 Visualizations

* Bar Chart → Investment by Sector
* Pie Chart → Return Distribution
* Scatter Plot → Risk vs Return
* Box Plot → ROI Distribution
* Line Chart → Monthly Trend

---

## 📊 Dashboard

All visualizations are combined into a single interactive dashboard using Plotly.
